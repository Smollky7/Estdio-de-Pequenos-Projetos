Jogo da Cobrinha

O projeto da cobrinha é um clássico desenvolvido em HTML e JavaScript. Ele apresenta uma cobrinha que cresce ao comer o alimento, com uma pontuação exibida na tela e a possibilidade de reiniciar o jogo. O design inclui efeitos visuais atraentes e modernos.

Gerenciador de Tarefas

Este projeto é uma aplicação web para gerenciar tarefas com uma interface intuitiva. Ele permite adicionar, completar e remover tarefas de forma fácil, oferece opções para personalizar o tema, e inclui um modal com informações sobre o aplicativo. A aplicação é responsiva e acessível, garantindo uma boa experiência em diversos dispositivos.

Chat Bot

Esta aplicação é um chat simples com um robô chamado Hobot, que responde automaticamente a mensagens com respostas pré-configuradas. A interface inclui um perfil do robô, uma área para mensagens e um formulário para envio de texto. O design é responsivo e moderno, ideal para demonstrações de interação básica com bots.

Books Dashboard

Esse é uma aplicação web interativa para gerenciar a leitura de livros. Oferece visualizações em lista e grade, mostra detalhes dos livros como título e progresso de leitura, e permite alternar entre modo claro e escuro. Inclui um botão para adicionar novos livros e exibe estatísticas sobre a leitura. O design é moderno e responsivo, proporcionando uma experiência de usuário agradável e intuitiva.

Calculator IOS

Este projeto é uma calculadora web que replica o design da calculadora do iOS 11. Ela possui um layout simples e funcional, com botões para operações básicas e uma interface limpa. Desenvolvida com HTML, CSS e JavaScript, a calculadora oferece uma experiência interativa com animações para os botões e suporta operações matemáticas essenciais.
